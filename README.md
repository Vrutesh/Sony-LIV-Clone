# Sony Liv Landing Page Project

This project is a replica of the landing page of Sony Liv, a popular streaming platform. The landing page is built using HTML, CSS, and JavaScript.

## Features

- Slider: Implementing a slider feature to showcase featured content or promotions.
- JavaScript Concepts: Various JavaScript concepts such as DOM manipulation, event handling, and animation are used.
- Clean and organized code structure.

## Technologies Used

- HTML
- CSS
- JavaScript
